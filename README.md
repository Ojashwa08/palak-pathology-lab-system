# 🧪 Palak Pathology Lab Management System

A console-based Java application designed to manage pathology lab operations using MySQL and JDBC.

---

## 🚀 Features
- 🔐 Admin Login System
- 👨‍⚕️ Patient Management (Add, Update, Delete, Search)
- 🧪 Test Management
- 📋 Assign Tests to Patients
- 📊 Report Generation
- 💰 Billing System (Auto Calculation)
- 🔍 Search & Filter

---

## 🛠️ Tech Stack
- Java (Core Java)
- JDBC
- MySQL
- Console-based UI

---

## 📂 Project Structure
com.pathology
- model
- dao
- service
- ui
- util
- main

---

## ⚙️ How to Run
1. Import project into Eclipse
2. Setup MySQL database using schema.sql
3. Update DB credentials in DBConnection.java
4. Run Main.java
